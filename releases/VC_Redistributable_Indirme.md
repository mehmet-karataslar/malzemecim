# Visual C++ Redistributable İndirme

Windows uygulamasının çalışması için Visual C++ Redistributable gerekebilir.

## İndirme Linkleri

### Visual C++ Redistributable 2015-2022 (x64)
- **Doğrudan İndirme:** https://aka.ms/vs/17/release/vc_redist.x64.exe
- **Microsoft Resmi Sayfası:** https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist

### Visual C++ Redistributable 2015-2022 (x86) - 32-bit sistemler için
- **Doğrudan İndirme:** https://aka.ms/vs/17/release/vc_redist.x86.exe

## Kurulum

1. İndirilen `.exe` dosyasını çalıştırın
2. Kurulum sihirbazını takip edin
3. Kurulum tamamlandıktan sonra bilgisayarı yeniden başlatın (gerekirse)
4. Malzemecim uygulamasını tekrar çalıştırın

## Not

Windows 10 ve 11'in çoğu sürümünde Visual C++ Redistributable zaten yüklüdür. 
Eğer uygulama çalışmıyorsa veya DLL hatası alıyorsanız yukarıdaki linklerden indirip yükleyin.
